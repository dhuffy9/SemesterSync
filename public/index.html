How to add a course:

  1. Click the New Course button at the top-left of the site (the + icon is the same button).
  2. Search the course by course number or class name; doing this autofills the rest of the inputs.
  3. To add a course manually, fill the fields after clicking New Course:
    1. Course Number
    2. Course Title
    3. Instructor
    4. Credits
    5. Days (Sunday — Saturday checkboxes)
    6. Start Time (HH:MM AM/PM)
    7. End Time (HH:MM AM/PM)
  4. Click Add Course to save the course.

If something doesn't save, try these steps: reload the page, clear your browser cache, ensure your browser allows localStorage/cookies, or try a different browser. For help, email djh61@pct.edu — include the steps to reproduce the issue.
